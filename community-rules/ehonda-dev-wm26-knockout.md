# Kicktipp Community Knockout-Stage Scoring Rules

These rules apply to knockout-stage matches whose match input uses the result basis `finalScoreIncludingExtraTimeAndPenaltyShootout`.

## Result Basis

- Predict the final score that Kicktipp will evaluate after the match has been decided.
- A match decided in regular time is evaluated with the regular-time score.
- Goals scored in extra time count toward the evaluated score.
- When a penalty shootout is played, every converted shootout penalty is added to the score after extra time.
- A draw is impossible in the final evaluated result because the match must produce a winner.
- Example: A match that is 2:2 after extra time and has a 5:4 penalty shootout is evaluated as 7:6.

## Scoring System

| Result Type | Tendency | Goal Difference | Exact Result |
|-------------|----------|-----------------|--------------|
| Win         | 2        | 3               | 4            |

* Tendency: Predicting the winner
* Goal Difference: Predicting the winner and the final evaluated goal difference
* Exact Result: Predicting the exact final evaluated score
